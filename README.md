# LauncherReleases
Releases for the Borion launcher

Minimum requirements: A 64 bit Windows 10/11 PC with a GPU (integrated or dedicated) that can run at least Vulkan 1.3 or OpenGL 3.3.

The launcher will prefer Vulkan over OpenGL, unless Vulkan 1.3 isn't supported or the user created an "opengl.txt" file in %localappdata%\BorionLauncher.

The directory of data used by the launcher (logs, downloaded Borion DLL) is %localappdata%\BorionLauncher.
